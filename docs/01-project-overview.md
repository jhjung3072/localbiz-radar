# LocalBiz Radar 프로젝트 개요

## 목적

LocalBiz Radar는 공개 가능한 공공 데이터를 활용해 지역 상권의 점포 분포, 업종 구성, 경쟁 밀도, 지역 비교를 보여주는 분석 대시보드입니다. 포트폴리오 관점에서는 서비스 방향, 보안 원칙, 프론트엔드 정보 구조, 백엔드 확장 가능성을 명확히 보여주는 것을 목표로 합니다.

## 제품 범위

현재 범위는 프로젝트 기반 구축과 초기 화면 구현입니다. 실제 공공 API 호출, 지도 SDK, 인증, LLM 기능은 포함하지 않습니다.

## 주요 화면

- 홈: 제품 소개, 핵심 기능, 데이터 출처와 API 키 보안 원칙 안내
- 대시보드: 업종별 분포 차트, 요약 지표, 인기 업종 순위, 상태 UI 예시
- 상가 목록: TanStack Table 기반 목업 목록, 키워드 검색, 업종 및 지역 필터
- 상권 분석: 지역과 업종 선택 UI, 경쟁 지표, 카테고리 구성, A/B 비교 미리보기
- API 설정: 공공 API 키 발급과 백엔드 프록시 패턴 안내

## 데이터 원칙

초기 화면의 값은 모두 목업 데이터입니다. 실제 연동 단계에서는 Spring Boot 백엔드가 공공 API를 호출하고, 프론트엔드는 백엔드 내부 API를 통해 가공된 결과만 받습니다.

## 제외 항목

- 실제 공공 API 통합
- 프론트엔드의 공공 API 직접 호출
- 지도 SDK
- 인증과 권한 관리
- LLM 또는 문서 협업 기능
