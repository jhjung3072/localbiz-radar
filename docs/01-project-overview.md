# LocalBiz Radar 프로젝트 개요

## 목적

LocalBiz Radar는 공개 가능한 공공 데이터를 활용해 지역 상권의 점포 분포, 업종 구성, 경쟁 밀도, 지역 비교를 보여주는 분석 대시보드입니다. 포트폴리오 관점에서는 서비스 방향, 보안 원칙, 프론트엔드 정보 구조, 백엔드 확장 가능성을 명확히 보여주는 것을 목표로 합니다.

## 제품 범위

현재 범위는 Store/Region/Analysis API, 지도 조회 API, CSV 동기화 API, 제한된 OpenAPI 동기화 API와 점포 목록, 대시보드, 상권 분석, 지도, 데이터 동기화 화면 연동입니다. 인증과 권한 관리는 포함하지 않습니다.

## 주요 화면

- 홈: 제품 소개, 핵심 기능, 데이터 출처와 API 키 보안 원칙 안내
- 대시보드: Analysis API 기반 업종별 분포 차트, 요약 지표, 인기 업종 순위
- 점포 목록: Spring Boot API 기반 목록, 키워드 검색, 업종 및 지역 필터
- 상권 분석: Analysis API 기반 지역/업종 선택, 경쟁 지표, 카테고리 구성, A/B 비교
- 지도: Kakao Maps 기반 점포 marker, 지역/업종 필터, 중심 좌표 반경 검색
- 데이터 동기화: 소상공인 상가정보 CSV/OpenAPI dry-run, 실제 반영, sync log 조회
- API 설정: 공공 API 키 발급과 백엔드 프록시 패턴 안내

## 데이터 원칙

점포 목록, 대시보드, 상권 분석, 지도 화면은 Flyway로 주입한 개발용 Store/Region seed data와 CSV/OpenAPI 동기화 데이터를 조회하거나 계산합니다. 현재 분석 지표와 반경 검색은 점포 수, 업종 분포, 위도/경도만 반영한 개발용 지표입니다. 공공 데이터 OpenAPI 호출은 Spring Boot 백엔드에서만 수행하고, 프론트엔드는 백엔드 내부 API를 통해 가공된 결과만 받습니다.

## 제외 항목

- 프론트엔드의 공공 API 직접 호출
- 전국 단위 대량 동기화
- 인증과 권한 관리
- 문서 협업 기능
